Setup > cd

cd

folder/reponame

Description

Use the cd command to navigate folders and files on your computer

Options

filename

You can type a folder or filename to navigate to it. You can also type multiple folders to navigate to a nested folder: folder1/nestedfolder/folder3

..

use cd .. to naviagte backwards. Like before, you can use multiple to navigate back up the folder tree: cd../../..
