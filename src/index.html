Setup > cd

cd

folder/reponame

Description

Use the cd command to navigate folders and files on your computer

Options

filename

You can type a folder or filename to navigate to it. You can also type multiple folders to navigate to a nested folder: folder1/nestedfolder/folder3

..

use cd .. to naviagte backwards. Like before, you can use multiple to navigate back up the folder tree: cd../../..

Projects > clone

git clone

--options

Description

Clones a repository into a newly created directory, creates remote-tracking branches for each branch in the cloned repository (visible using git branch -r), and creates and checks out an initial branch that is forked from the cloned repository’s currently active branch.

Options

--branch

Instead of pointing the newly created HEAD to the branch pointed to by the cloned repository’s HEAD, point to branch instead. In a non-bare repository, this is the branch that will be checked out. --branch can also take tags and detaches the HEAD at that commit in the resulting repository.
